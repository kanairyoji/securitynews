1. CVE-2026-26142 — Nuance PowerScribe RCE（医療インフラ）

出典: CrowdStrike Blog（2026-06-14）

  • CVSS 9.8（Critical）/ CWE-502: デシリアライゼーションによる RCE
  • Nuance PowerScribe は医療施設で広く使われる放射線レポート・ワークフロープラットフォーム
  • 認証不要のリモート攻撃者がコード実行可能。ユーザー操作不要・低攻撃複雑性
  • 医療データや臨床インフラの侵害につながる可能性

2. CVE-2026-44815 — Windows DHCP クライアント RCE（全 Windows 端末）

出典: CrowdStrike Blog（2026-06-14）

  • CVSS 9.8（Critical）/ CWE-121: スタックベースバッファオーバーフロー
  • 不正 DHCP サーバーを同一ネットワークに設置することで、DHCP リクエストに応答する脆弱クライアントをリモート RCE
  • OT/ICS 環境や工場・制御システムのネットワーク内での悪用リスクが高い
  • 緩和策：DhcpGetOriginalSubnetMask API を呼び出すアプリを監査・制限

3. CVE-2026-47291 — HTTP.sys Critical RCE（公開 Web サービス）

出典: CrowdStrike Blog（2026-06-14）

  • CVSS 9.8（Critical）/ CWE-190 + CWE-122: 整数オーバーフロー＋ヒープバッファオーバーフロー
  • IIS や Windows Web サービスを公開しているサーバーが対象
  • 認証不要・ユーザー操作不要でリモート RCE
  • 緩和策：MaxRequestBytes レジストリ値を 65,534 バイト以下 に設定

4. BWH Hotels（Best Western等）データ侵害

出典: DataBreaches.net（2026-06-14）

  • ホテル宿泊客の個人情報が 6 ヶ月間にわたり 侵害
