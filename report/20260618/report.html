地政学リスク・国家サイバーオペレーション

カテゴリ: カテゴリ1「地政学リスク・国家サイバーオペレーション」
対象日: 2026年06月18日
生成日時: 2026-06-19

総評

英国NCSCは、英国の重要社会基盤（CNI）に対するサイバー攻撃の75%が敵対的国家主体によるものだと警告を発した。Fortinet機器を標的にした「FortiBleed」キャンペーンは7万3000台以上の機器の認証情報を流出させ、NCSCはFortinet製品使用組織に緊急対応を求めた。Evil Corp連携のSocGholishボットネットに対する国際的な法執行作戦が展開され、約1万5000のWordPressサイトがクリーンアップされた。DragonForceランサムウェアはMicrosoft Teamsを悪用してC2トラフィックを偽装する高度な手法を採用しており、国家・犯罪組織双方の攻撃高度化が顕著だ。

1. NCSCが英国CNIへの国家サイバー攻撃を警告

出典: Infosecurity Magazine（2026-06-18）

英国NCSCのCEO Richard Horneは、英国重要インフラへのサイバー攻撃の75%が国家主体によるものであると発表。同時に、Fortinet製ファイアウォール・VPNゲートウェイを対象とした世界規模キャンペーンへの対応を呼びかけるアラートを発出した。

2. FortiBleed：7万5000台のFortinet機器の認証情報漏洩

出典: BleepingComputer / Security Affairs / CSO Online（2026-06-18）

「FortiBleed」と呼ばれるキャンペーンで、FortinetおよびFortiGate VPN機器7万3932台（別報では7万5000台）の管理者パスワードが公開サーバー上に晒された。NCSCはFortinet使用組織に対して即時対応を要求。LATAM地域でもFortinet・Ivanti製品の脆弱性悪用（Operation Escaneo）が報告されている。

3. Evil Corp連携SocGholishボットネット、国際法執行で壊滅的打撃

出典: BleepingComputer（2026-06-18）

国際法執行機関の共同作戦により、SocGholishマルウェアに感染した約1万5000のWordPressサイトがクリーンアップされ、ロシアのサイバー犯罪グループEvil Corpに関連する100台以上のサーバーが摘発された。

4. DragonForce、Microsoft Teams経由でC2を隠蔽

出典: The Hacker News（2026-06-18）

DragonForceランサムウェアグループは、カスタム製Go言語RAT「Backdoor.Turn」を使用し、Microsoft TeamsリレーサーバーへのC2トラフィックを隠蔽する手法を採用していることが判明した。正規サービスの悪用による検知回避が進んでいる。

5. MicrosoftによるWindowsクリッパーマルウェアキャンペーンの詳細公開

出典: The Hacker News / BleepingComputer（2026-06-18）

Microsoftは2026年2月以降、USB LNKワームとTorベースC2を組み合わせた暗号通貨クリッパーキャンペーンの詳細を公開。Windows Script HostとActiveXを悪用し、感染拡大とウォレット情報窃取を同時に行う手口が明らかになった。
