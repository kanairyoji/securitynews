カテゴリ: カテゴリ1「地政学リスク・国家サイバーオペレーション」
対象日: 2026年06月18日
生成日時: 2026-06-19

総評

英NCSCが、過去1年に英重要インフラを標的とした200件のインシデントの75%がロシア・中国・イランなどの国家系アクターに起因すると公表し、Volt Typhoon型の「事前展開（prepositioning）」を最大の脅威例として挙げた。台湾周辺では中国海警・海上民兵による「準検疫（quasi-quarantine）」グレーゾーン作戦が観測され、PLAは2027年目標を意識した「戦場執行能力」を強調。中朝首脳会談では国防相同士が同席し軍事協力深化が確認された。FortiBleedではロシア語話者の多人数攻撃グループが約75,000台のFortiGateから認証情報を窃取し、トルコのNATO防衛請負業者から機密文書を盗んだとされ、犯罪と国家オペレーションの境界が曖昧化している。

1. 英NCSC: 重要インフラ攻撃の75%が敵対国家に起因

出典: Infosecurity Magazine（2026-06-18）
Richard Horne CEOがRUSI講演で、2025年6月〜2026年5月のCNI関連200件のインシデントの3/4がロシア・中国・イラン等の国家系アクターに起因すると公表。脅威空間を「far / mid / near」の3層で整理し、midでは攻撃者がクラウド・OSSサプライチェーンを悪用、クラウドAIサービスが今後攻撃を加速すると警告。Volt Typhoonによる米インフラ事前展開を最も顕著な例とし、「2028年までにAI能力が英CNIのレガシー脆弱性に対して使われる公算が高い」とのNCSC評価を引用。「明日の紛争の物理的標的選定は今日収集された情報に基づく」と述べた。

2. 台湾を巡る中国のグレーゾーン作戦と2027年目標

出典: Jamestown Foundation / OODA Loop（2026-06-18）
Jamestownは台湾東方の海警・海上巡視活動を「準検疫」モデルの試験と分析。物理的封鎖を伴わず「検疫が可能」との外観を作るグレーゾーン戦術と評価。台湾外相は中国による国際イベント参加妨害圧力が「新常態」になったと述べた。別稿ではPLA Dailyの「戦場執行」議論が2027年目標（前線指揮官による上位作戦意図の実行能力）を裏付けると分析した。

3. FortiBleed: ロシア語話者グループの大規模認証窃取オペレーション

出典: BleepingComputer（2026-06-18）
研究者Bob Diachenkoが、ロシア語話者の多人数グループによる作戦を特定。FortiGate SSL VPN 320,777標的に約11.6億回、MSSQL 163,650システムに約21億回の認証試行を実施し、傍受したVPN認証ハッシュを45-GPUクラスタ（Hashtopolis管理）で解読、AD環境へ横展開。日本・台湾・ベトナム・イラク・トルコの組織が完全侵害され、トルコのNATO防衛請負業者から機密文書が盗まれたとされる（詳細はカテゴリ6・8参照）。

4. 中朝首脳会談 — 軍事・対米連携の演出

出典: The Cipher Brief（2026-06-18）
習近平が2019年以来の訪朝。中朝両国防相（董軍・盧光鉄）も会談に同席し、軍事・法執行交流の強化を確認。「抗米精神」を強調し、北朝鮮は事実上の核保有国としての承認と威信を獲得。寧辺の新ウラン濃縮施設稼働は対米メッセージと分析される。

3. 40年来の連邦サイバー政策の失敗

出典: The Cipher Brief（2026-06-18）
元FBIサイバー部門副次官補が下院国土安全保障委で証言し、連邦政府はランサムウェアをテロ行為として指定することを検討すべきと主張。議会が長年取り組まずにきた構造的課題を指摘した。

4. EO 14409後のAI安全保障とAIモデルの政府規制

出典: CrowdStrike Blog / Risky Business（2026-06-18）

5. EUが6Gセキュリティ「Shield-6G」で先行

EU主導のShield-6Gは、AI脅威検知・デジタルツイン・ハニーポット等を統合し6G網を将来の脅威から守るフレームワーク。技術標準の主導権確保を狙う。
