総評

北朝鮮APTグループKimsukyによるKimJongRAT進化の最新分析がIIJから公開され、GitHub等の正規サービスを悪用したLOTS（Living off Trusted Sites）戦術の高度化が確認された。ロシアが自国の人権活動家に対しCellebrite製品の提供契約終了後もその技術を用いて端末解析を行っていたことが判明し、サンクション迂回と技術残留の問題が浮上した。NSA・Five Eyes連合は「AI支援のサイバー脅威により、攻撃タイムラインが数年から数ヶ月に圧縮されている」と緊急警告を発し、防衛コミュニティに対応加速を求めた。ウクライナでは国営郵便局Ukrposhताがサイバー攻撃を受けてアプリサービスが停止し、ロシア・ウクライナ間のサイバー戦が継続していることが確認された。

1. KimJongRAT：LOTSを活用した継続的進化（北朝鮮・Kimsuky）

出典: IIJ Security Diary（2026-06-25）

2026年5月、北朝鮮帰属APTグループKimsuky（Earth Kumiho）が使用するKimJongRATがGitHub等の正規サービスを悪用してC2通信を行う「LOTS（Living off Trusted Sites）」戦術を採用したキャンペーンが観測された。
- KimJongRATはInfoStealerとRATの両機能を持つマルウェア（2013年頃から使用継続）
- GitHubを悪用したC2通信はセキュリティ製品による検知回避を目的とする
- 同グループは日韓の政府・研究機関・防衛関連組織を標的とする傾向がある

2. ロシア：Cellebrite提供停止後も使用継続

出典: CyberScoop、The Record、TechCrunch（2026-06-25）

ロシア当局が2021年3月の提供停止宣言後もCellebriteの端末解析ツールを使用し続け、国内の人権活動家のiPhoneを解析したことがセキュリティ研究者の調査で判明した。
- Cellebriteはロシアへの販売停止を表明していたが、既存デバイスの回収・無効化ができなかった
- 法執行機関向けに提供されたFDES（Forensic Device Extraction System）が転用された
- 制裁・輸出規制の実効性とデュアルユース技術の残留リスクを示す事例

3. NSA/Five Eyes：AI脅威タイムラインが「数ヶ月」に圧縮

出典: Cyber Defense Magazine（2026-06-25）、Risky Business（2026-06-25）

NSAがFive Eyes（英米加豪NZ）のサイバーセキュリティ機関と連名で緊急警告を発出。AIの普及により、高度なサイバー攻撃に必要な時間が従来の「数年」から「数ヶ月」に短縮されたと警告した。
- オープンウェイトモデルの普及が攻撃者の参入障壁を大幅に下げている
- 国家支援グループに加え、非国家アクターによる高度攻撃が増加する見込み
- Risky Businessはこの警告を「Mythos議論の終焉」として分析

4. ウクライナ国営郵便局Ukrposhタがサイバー攻撃を受けサービス停止

出典: The Record、DataBreaches.net（2026-06-25）

ウクライナの国営郵便サービスUkrposhタが6月25日未明のサイバー攻撃によりモバイルアプリ等のサービスに障害が発生した。攻撃主体は不明。

- ロシアの乳製品メーカー（バシコルトスタン共和国）でも同日サイバー攻撃による業務妨害が報告されており、双方向でのサイバー戦が継続
