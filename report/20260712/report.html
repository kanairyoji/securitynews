出典: Security Affairs（2026-07-12）

出典: Security Affairs（2026-07-12）

2. AI モデルの「行政的キルスイッチ」— 法的根拠は未検証のまま

出典: Habr（2026-07-12）

ロシアの技術系メディア Habr が、約3週間にわたる AI モデル「Fable 5」の世界規模の停止措置から復帰までの全経緯を検証した。記事によれば、Anthropic は米政府との交渉を経てモデルの防護機構を改修し、輸出規制の解除を得て提供を再開したが、より高性能とされる上位モデルは非公開のまま残された。

論点として同記事が強調するのは以下の点である。

論点 内容

3. 欧州委員会、NIS2 未施行の加盟国を提訴

4. 「Chat Control 1.0」がストラスブールで再浮上
